> **Version**: 0.1 (2026-06-24)

## 概要

DTMプラグイン・音源・機材のセール情報をキュレーションするページ。
「AI Digest by Reina」と同じリポジトリに同居し、デザイン言語を共有する。

**コンセプト**: 「今買わないと損するやつ、ぱっと見でわかる」

## ページ構成

### 1. ヒーロー（ヘッダー）

- タイトル: **DTM Deals by Reina** 🎹
- サブタイトル: 「セール情報、うちがまとめたよ」
- 更新日時表示
- AI Digest ↔ DTM Deals の相互リンク

### 2. カレンダービュー（メインコンテンツ）

**ソース: DAWHACK (dtm-sale.com)**

終了日ベースでセールを一覧表示。DAWHACKの「開催中＆終了間近」ページの構造をそのまま活かす。

```

│  ├ ...                                       │
│                                              │

```

#### カードに含む情報

| フィールド | ソース | 例 |
|-----------|--------|-----|
| 製品名 | DAWHACK | 「Mix Monolith」 |
| メーカー名 | DAWHACK | AYAIC |
| 割引率 | DAWHACK | 72%OFF |
| 元値 → セール価格 | DAWHACK | 43,401円 → 12,026円 |
| セール終了日 | DAWHACK | 6月24日 |
| カテゴリタグ | 自動分類 | 🎹シンセ / 🎛️エフェクト / 🎸ギター / 🎤ボーカル / 🥁ドラム / 🎵DAW |
| 購入リンク | DAWHACK記事経由 | dtm-sale.com/XXXXX/ |
| レビュー記事リンク | DTMステーション（あれば） | dtmstation.com/archives/XXXXX |

#### 表示ルール

- **終了日が近い順に表示**（今日・明日終了は赤ラベル「⚠️ 終了間近！」）
- **残り3日以内**: 赤背景でハイライト
- **残り7日以内**: オレンジラベル
- **7日以上先**: 通常表示
- **終了済みセール**: 非表示（or グレーアウト + 取り消し線、最大3日間残す）
- **割引率50%以上**: 🔥 バッジ付き

### 4. Reinaざっくりレビュー

**全セールアイテムに、Reinaのざっくり解説を付与する。**

DTMステーションのレビュー記事がない製品でも「これ何に使うの？」がわかるように、以下の観点で1-2行のコメントを付ける:

- **何ができるか**: 「アンプシミュ」「コンプレッサー」だけじゃなく、具体的な用途
- **どの工程に効くか**: トラック作成 / 打ち込み / 録音 / マスタリング
- **こういう悩みに**: 「音圧上げたいけど潰したくない人向け」等
- **ジャンル適性**: テクノ向き / メタル向き / オールジャンル 等

```
🎛️ Mix Monolith — トラックバランスをAIが自動調整。
   ミックス沼にハマりがちな人に。マスタリング前の下ごしらえ用。

🎸 BIAS X — AIプロンプトでギタートーンを生成。
   メタルのリフ録りからクリーンまで。アンプ実機持ってない人の救世主。

🎹 Serum 2 — ウェーブテーブルシンセの王様。
   テクノ/EDMのリード・ベース・パッド全部いける。迷ったらこれ。
```

注目セール（割引率70%以上、有名製品など）には追加で買い判定コメント:
```
💬 Reina: 「年1回あるかないかレベルの割引。迷ったら買い！」
```

## データ取得方式

### Phase 1（MVP: 静的HTML生成）

AI Digest と同じ方式。Heartbeat or 手動でスクレイピング → HTML生成 → git push。

#### ソースサイト一覧（ジャンル得意分野付き）

| 優先度 | サイト | URL | 得意ジャンル | 取得方式 |
|--------|--------|-----|------------|---------|
| 🥇 メイン | DAWHACK | dtm-sale.com/44766/ | **オールジャンル**。終了日ベースの構造化データ | Playwright スクレイピング |
| 🥈 レビュー | DTMステーション | dtmstation.com/feed | **レビュー・解説記事**。新製品紹介に強い | RSS |
| 🥉 補完 | DTMer.info | dtmer.info/feed/ | **網羅性No.1**。無料配布・フリープラグインに強い | RSS |
| 📖 参考 | 誰得ギターDTM | guitar-type.com | **ギター系プラグイン特化** | 手動参照 |
| 📖 参考 | 櫻井徳右衛門.jp | sakutoku.jp | **月次まとめ・マスタリング系レビュー** | 手動参照 |
| 📖 参考（英語） | Plugin Boutique | pluginboutique.com/deals | **海外セール直**。日本語サイトで未カバー分の補完 | 手動参照 |
| 📖 参考（英語） | rekkerd.org | rekkerd.org/deals/ | **無料配布・インディーメーカー発掘** | 手動参照 |

1. **DAWHACK スクレイピング**
   - URL: `https://dtm-sale.com/44766/`（開催中＆終了間近ページ）
   - Playwright で取得 → 終了日ごとのセクションをパース
   - 各アイテムから: 製品名、メーカー、割引率、価格、終了日、リンクを抽出

2. **DTMステーション RSS**
   - URL: `https://www.dtmstation.com/feed`
   - セールカテゴリの記事をフィルタ（category に「DTMセール情報」を含むもの）

3. **DTMer.info（補完用）**
   - RSS: `https://dtmer.info/feed/`
   - DAWHACK に載ってないセールがあれば補完

### Phase 2（将来: 自動更新）

- Heartbeat で毎日1回、スクレイピング → HTML再生成 → auto-push
- セール終了間近アラートをDiscord DMに送信

## デザイン

AI Digest by Reina と同じデザイン言語:

- **カラー**: ライトテーマ（白ベース + パステル紫/ピンク）
- **フォント**: Noto Sans JP
- **カード型レイアウト**: 角丸14px、影あり
- **レスポンシブ**: モバイルファースト
- **ダークモード**: 将来対応（Phase 2）

### DTM Deals 固有のデザイン要素

- **緊急度バッジ**: 残日数に応じた色分け（赤/オレンジ/通常）
- **割引率バッジ**: 大きめフォント、目立つ配色（🔥50%以上）
- **価格表示**: 元値に取り消し線 → セール価格を太字・アクセントカラー
- **カテゴリアイコン**: 絵文字ベース（上記表参照）
